# 中古せどりチェックリスト 公開用

## 使い方

- `index.html` を静的サイトとして公開する。
- スマホでは公開URLをブックマーク、またはホーム画面に追加して使う。
- 内容を更新したら `index.html` を差し替えて再公開する。
- 画面右上の「更新」ボタンで、スマホ側のキャッシュを避けて読み直せる。

## 推奨公開先

- GitHub Pages: 更新履歴を残しやすく、今後の修正を反映しやすい。
- Netlify / Cloudflare Pages: ファイルを置くだけでも公開しやすい。

## 注意

- 公開URLを知っている人は閲覧できるため、仕入れノウハウを完全に非公開にしたい場合は、パスワード付きの公開先を使う。
